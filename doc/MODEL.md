# 数据模型

## 菜单管理(`menu`)

| 字段       | 中文说明   | 字段类型 | 备注                                               |
| ---------- | ---------- | -------- | -------------------------------------------------- |
| id         | 自增 ID    | 数值     | 主键                                               |
| record_id  | 记录 ID    | 字符串   | uuid                                               |
| code       | 菜单编号   | 字符串   |                                                    |
| name       | 菜单名称   | 字符串   |                                                    |
| type       | 菜单类型   | 数值     | 10：系统<br/> 20：模块 <br/>30：功能 <br/>40：动作 |
| sequence   | 排序值     | 数值     |                                                    |
| icon       | 图标       | 字符串   |                                                    |
| link       | 跳转链接   | 字符串   |                                                    |
| level_code | 分级码     | 字符串   |                                                    |
| parent_id  | 父级 ID    | 字符串   |                                                    |
| status     | 状态       | 数值     | 1:启用<br/> 2:停用                                 |
| created    | 创建时间戳 | 数值     |                                                    |
| creator    | 创建人     | 字符串   |                                                    |
| deleted    | 删除时间戳 | 数值     |                                                    |

| created   | 创建时间戳 | 数值     |                    |
| creator   | 创建人     | 字符串   |                    |
| deleted   | 删除时间戳 | 数值     |                    |
